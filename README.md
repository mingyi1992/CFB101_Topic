# CFB101_Topic

## 功能 : 
    (1)利用Yolo模組辨識圖片中的菜色並將其營養資訊回傳
    (2)根據使用者提供的資訊推薦一份使用者可能會喜歡的菜單
    
### 注意事項 : 若要自行訓練模組，則yolo要用的圖片在label時須注意以下幾點
### (1)路徑應避免使用中文(2)label的標籤切勿使用空白及中文(3)無需事前resize照片
  
## 文件內容 : 
    包含資料的網路爬蟲在內的各種工具檔，由於檔案大小/照片資料版權等問題，孰不提供照片檔本身
